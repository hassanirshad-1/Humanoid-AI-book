## Who This Book Is For
You do **not** need a PhD. You do **not** need to be a math genius.
You just need:
1.  **Curiosity**: A desire to know how things work.
2.  **Persistence**: Robots (even simulated ones) can be frustrating. They fall over. A lot.
3.  **A Computer**: Any basic Mac, Windows, or Linux machine will do.

## How to Read This Book
This is a **"Concrete Sandwich"** book.
1.  **The Hook**: We start with a cool result or problem.
2.  **The Intuition**: We explain *why* it works using simple analogies (no math yet!).
3.  **The Code**: We write real, running code.
4.  **The Math**: We sneak in the formal equations *after* you understand the intuition.
